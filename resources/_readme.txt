Minecraft 1.6.4　MinecraftForge 9.11.1.916　BuildCraft 4.1.2　アドオン

BuildCraftアドオンCompactEngine-4.1.2　圧縮木エンジンMOD

概要
	木エンジンを圧縮して出力をアップさせた、圧縮エンジンを追加する。
	ピストン速度は木エンジンの二倍。
	最終的に赤ピストンで固定され、一定時間後に爆発する。
	爆破耐性は通常の木エンジンより高く、丸石と同じぐらい（木エンジンは土と同じ）


追加される機器の設定

	圧縮木エンジン（CEW2-0008～CEW2-2048）
		BC社が、独自の木圧縮技術を持つCE社と提携して開発した、高出力なクリーンエンジン。
		木エネルギーパイプを補強する機能が備わっており、エネルギー流出のロスを抑えることに成功した。
		しかし製作コストが非常に高いため、BC社の機器を使いこなしている技師でなければ扱いは難しい。

		BC3.2.0世代への正式採用は見送られたが、CE社が単独で開発を決行。
		BC社の仕様変更に対応した結果、排熱処理が追いつかずに、熱暴走して爆発するようになった。
		また爆発力を抑える機構も脆弱になったため、爆発の規模も増大した。
		クリーンエンジンでありながら、IC社製の核爆弾すら凌駕する爆発力を持つこのエンジンは、
		皮肉を込めて「クリーンな兵器」と呼ばれている。

	エネルギー計測器（CEC-03）
		CE社製の計測器。BC社のエンジン、マシン、パイプのエネルギー量を計測する。
		パイプは瞬間的なエネルギー量しか測定できないため、時間当たりの流量を調べるには向かない。
		エンジンの出力は、現在のピストン速度から算出される理論値である。
		C-01ではエンジンの熱量しか計測できなかったが、C-02にてパイプやマシンにも対応した。
		BC3.2.0世代ではマシン温度の計測方法も基準化されたため、C-03では温度測定も可能となっている。

エネルギー計測器の使い方

	BuildCraft機器のエネルギー（MJ）量を計測する機器。
		機器に当て続ける事により、4tickサイクルで計測できる。

	エンジンを右クリックすると、出力、熱量、温度を表示する
		出力はピストンの色に応じた最大出力理論値。石と鉄では生産能力が低いため、実際にこの数字は出ない。
		熱量は蓄積されているMJ量。
		熱量か温度のどちらかが限界を超えると爆発する。
		（ただし、圧縮木エンジンは自身の動作で熱量が限界になっても爆発しない）

	BCのマシンをクリックすると、熱量と受入量を表示する。
		熱量はエンジンに同じ。熱量が許容限界になると、それ以上のエネルギーを受け付けなくなる。
		受入は1回の入力ごとに受け入れる事の出来る熱量の限界。

	エネルギーパイプを右クリックすると、瞬間エネルギー流量を表示する。
		クリックした瞬間の流量なので、やや正確さに欠ける。
		なお瞬間流量は、パイプ内のエネルギーの太さと比例する。

導入方法
	Minecraft 1.6.4
	MinecraftForge 9.11.1.916 ～ 
	BuildCraft 4.1.2
	以上が導入されている状態で、modsフォルダに入れる


設定ファイル
	CompactEngineId = 1529			圧縮木エンジンのID
	EnergyCheckerId = 19500			エネルギー計測器のID、32000にすると追加しない
	Add high compact engine = false	trueにすると512倍、2048倍の木エンジンを追加する
	CompactEngineExplosionPowerLevel = 1	エンジンの爆発力レベル。0=低い、1=標準、2=災害、3=崩壊
	CompactEngineExplosionTimeLevel = 1		エンジン爆発までの時間。0=長い、1=標準、2=短い、3=無理
	CompactEngineExplosionAlertMinute = 3	爆発までの残り時間が残りn分になったらアラートを表示する。
											0を指定すると、アラートを表示しない。
  OilFlowingSpeed     オイルの流れやすさ。0で水と同じ。20以下を指定で消えやすくなる。

使用上の注意
	512倍、2048倍圧縮木エンジンはエネルギー流量が多くエネルギーパイプが爆発しやすいので注意。
	またエンジン自体も爆発しやすく、更に爆発力が非常に高い。
	512倍、2048倍圧縮木エンジンを使用するには、設定ファイルから有効にする必要がある。
